Hostel Allotment App

It is a Hostel Allotment Application developed with Java , Node.js , Express and MongoDB as a backend.It authenticate the user with OTP sent through NodeMailer. This Application is used to allot the rooms to the students where students can select their hostels and rooms using Room matrix.
